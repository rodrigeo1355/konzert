# Konzert · Cobalto Edition — v2.0

Identidad propia de color para Konzert. Reemplaza la paleta verde anterior por **Cobalto**: un degradé tech/frío, único y diferenciado del resto de apps de música/eventos.

## Cambios

### Color
- Nuevo gradient principal: `#1e3a8a → #06b6d4` (135°)
- Escala completa Cobalto 200–900 en `colors_and_type.css`
- Glows decorativos actualizados (`--color-glow-cobalt`, `--color-glow-deep`, `--color-glow-cyan`)
- Tokens semánticos `--accent`, `--accent-hover`, `--accent-deep`, `--border-focus` apuntan a Cobalto
- Agregado `--color-cream: #f5f0e6` para versiones claras
- Agregado `--radius-xl: 1.5rem`

### Logo (base 07 · Radar)
- `assets/konzert-logo.svg` — isotipo: 4 círculos concéntricos en degradé Cobalto sobre fondo `#0a0a0a` con radio 46px
- `assets/konzert-wordmark.svg` — wordmark sobre oscuro
- `assets/konzert-wordmark-light.svg` — **nuevo** wordmark sobre crema (`#f5f0e6`)
- `assets/favicon.svg` — favicon 32×32 con el radar

### Preview
- `Konzert Design System.html` — **nuevo** preview completo del DS: hero, escala de color, type specimens, componentes (buttons, pills, inputs, event cards) y mockup de home
